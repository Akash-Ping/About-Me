Let Us C

This is an awesome book if you want to start learning programming. Each and every topic and concept is explained in a easy manner. The author of this book is Yashavant Kanetkar who is a computer science author of Indian origin.

What does it contain ?

It contains basics of C language, comprehencive introduction to fundamental concepts and explains them in a easy way that a beginner can understand easily.

My thoughts about it.

Let Us C was my first CS book that I read when I started coding, it helped me to understand basic concepts and funamental of a C Program. The Author taught through examples which made it more easy to understand

Operating System Princliples

Operating System Principles by Wiley is an excellent book for the concept of Operating System. It is divided into eight parts, each part explains different topics like first part Operating System ,second part process management, etc.

What does it contain ?

It contains important topics from the rapidly changing fields of operating systems and concepts like process-concept, multi-threaded programming, process scheduling ,etc.

My thoughts about it.

It is a very comprehensive guide to the workings of the computer / operating system with lots of excellent diagrams and good explanations of quite complex topics.

Introduction to Algorithms

Introduction to Algorithms is one of the best book for Data Structure and Algorithms.The author of this book is Thomas H. Cormen who is the Professor and former Chair of the Dartmouth College Department of Computer Science .

What does it contain ?

It contains comprehensive introduction to fundamental concept an a full spectrum of modern algorithms explained in a easy way.

My thoughts about it.

This book is impressive! It covers a lot of subject matter and is clearly worded. You'll be pushed into analyzing algorithms for theoretical data structures that you will learn concrete approaches to determining approaches.
